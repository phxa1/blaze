## 功能特性

- 高性能架构
- 易于扩展

## 快速开始

### 环境要求

- Node.js >= 14.0.0
- npm 或 yarn

### 安装

```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd blaze

# 安装依赖
npm install
```

### 使用

```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
